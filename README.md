Finance, Mathematics, Software Library
======================================
A modern C++ library for financial analytics

Directory Structure
-------------------

* <ROOT>
** bin - executables
** config - makefiles
** include - headers
** lib - libraries
** src - source code
*** <projects...>

## Building
{{
$ cd src
$ . .fmsrc
$ make
$ make test
$ make install
}}

!! Using
Set the include directory to {{<ROOT>/include}} and library directory to {{<ROOT>/lib}}. The current library is header only.
